# BunnyEra Free Open Source Collection

This repository collects **free, open-source, no-cost, simple, and brand-usable tools**  
that can directly support the BunnyEra ecosystem:

- BunnyEra-AI（AI / Agents / LLM）
- BunnyEra Console（系统工具）
- BunnyEra NotifyCenter（通知系统）
- BunnyEra Vision（视频 / 内容）
- BunnyEra Matrix（安全 / 隐私）

All tools in this repository are:
- 100% Free
- Open Source
- No subscription
- No hidden fees
- Easy to use
- Useful for BunnyEra brand development
