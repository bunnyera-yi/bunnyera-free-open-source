# Security / Privacy — Free & Open Source

## Core Tools
- Bitwarden — 开源密码管理
- Passbolt — 团队密码管理
- Cryptomator — 文件加密
- Outline VPN（自建）— 私有 VPN

## Extended Tools
- WireGuard — 最强 VPN 协议
- Pi-hole — 广告过滤
- Authelia — 单点登录
- Vaultwarden — Bitwarden 自托管
- KeePassXC — 本地密码管理
- OpenSnitch — 应用防火墙
- Caddy — 自动 HTTPS 服务器
- Traefik — 反向代理
