# Productivity / Vision Tools — Free & Open Source

## Core Tools
- Cap — 开源屏幕录制
- Screenity — 浏览器录屏
- Shotcut — 开源视频编辑
- Obsidian 插件生态 — 免费、强大

## Extended Tools
- Kdenlive — 专业视频编辑
- HandBrake — 视频压缩
- GIMP — 图像编辑
- Inkscape — 矢量图
- Audacity — 音频编辑
- ShareX — 截图 + 录屏
- MuseScore — 音乐制作
- Penpot — 开源设计工具
